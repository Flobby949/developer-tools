# 🛠️ 开发者工具集

🎆 **一个集成化的在线工具平台，为开发人员提供常用的代码工具和转换功能。**

> 🤖 **特别说明**: 本项目由 **Qoder** 全程辅助开发完成，从项目架构设计到代码实现、UI设计、功能优化，皆由AI智能生成。展示了AI在现代前端开发中的强大能力和实用性。

🌐 **在线体验**: [访问演示地址](https://www.flobby.top/tools)

## 🎆 项目特色

- 🤖 **AI全程开发** - 由Qoder全程辅助开发，展示AI在前端开发中的工程化能力
- 🎨 **现代化界面** - 基于Vue 3 + TypeScript构建，支持深色/浅色主题切换
- 📱 **响应式设计** - 完美支持桌面端和移动端使用
- ⚡ **实时处理** - 所有工具都支持实时转换和预览
- 🛠 **功能丰富** - 涵盖JSON、YAML、编码转换、正则表达式等常用开发工具
- 💾 **数据安全** - 所有处理均在本地进行，不上传任何数据到服务器
- 🌐 **无网可用** - PWA支持，离线后仍可正常使用

## 🔧 功能特性

### 🤖 AI智能对话

- **AI对话助手** - 集成先进的AI对话能力，支持代码生成、技术咨询、问题解答
- **Markdown渲染** - 支持代码高亮、数学公式、表格等丰富格式
- **对话历史** - 自动保存对话记录，支持导出和管理
- **智能建议** - 提供相关问题推荐和上下文理解

### 🔒 加解密工具

- **AES加解密** - 支持多种加密模式（CBC/ECB/CFB/OFB/CTR），多种填充方式，128/192/256位密钥长度
- **RSA加解密** - 非对称加密，支持1024/2048/4096位密钥，密钥对生成功能
- **MD5哈希** - 文件完整性验证，密码存储，支持大小写输出
- **SHA哈希** - 支持SHA1/SHA256/SHA512算法，区块链级别安全性

### 🔗 通信测试工具

- **WebSocket测试** - 实时连接测试，消息发送接收，连接状态监控
- **MQTT测试** - 支持MQTT协议测试，主题订阅发布，QoS级别设置
- **实时日志** - 详细的连接日志和消息记录

### 📄 文件处理工具

- **Base64文件转换** - 文件与Base64相互转换，支持预览和下载
- **文件类型检测** - 智能识别文件MIME类型
- **拖拽上传** - 便捷的拖拽文件上传体验

### 📱 二维码工具

- **二维码生成** - 支持文本、URL、WiFi信息等多种类型
- **二维码解析** - 支持图片上传、摄像头扫描、粘贴图片
- **高级配置** - 颜色自定义、纠错级别、尺寸设置
- **批量处理** - 支持批量生成和解析

### 🕐 时间处理工具

- **时间戳工具** - Unix时间戳与日期时间相互转换
- **多格式支持** - 支持秒级和毫秒级时间戳
- **时区处理** - 支持不同时区的时间转换

### 🔐 JWT解析工具

- **JWT解码** - 解析JWT令牌的Header、Payload和Signature
- **格式化显示** - 美化的JSON格式展示
- **有效性验证** - 检查JWT格式的正确性

### 🎨 颜色工具

- **调色盘工具** - 颜色选择器，RGB/HSL/HEX格式转换
- **颜色搭配** - 提供和谐的颜色搭配建议
- **取色器** - 从图片中提取颜色

### 📊 实体类Mock数据

- **Mock数据生成** - 根据Java/TypeScript实体类生成测试数据
- **智能解析** - 自动识别字段类型和约束
- **数据定制** - 支持自定义数据规则和格式

### JSON工具

- **JSON格式化** - 将JSON字符串格式化为易读的格式，支持自定义缩进
- **JSON压缩** - 去除JSON中的空白字符，压缩为一行
- **JSON转实体类** - 将JSON转换为多种编程语言的实体类定义
  - 支持Java、TypeScript、C#、Python
  - 自定义类名
  - 自动生成getter/setter方法(Java)

### YAML工具

- **YAML与Properties互转** - 在YAML格式和Java Properties格式之间转换
- **双向转换** - 支持YAML→Properties和Properties→YAML
- **示例数据** - 提供常用配置示例，一键加载测试

### 编码转换

- **URL编码/解码** - 对URL进行编码和解码操作，处理中文和特殊字符
- **Base64编码/解码** - 对文本进行Base64编码和解码
- **批量处理** - 支持大文本的编码转换
- **错误提示** - 智能识别无效输入并给出明确提示

### 正则表达式测试

- **实时匹配** - 输入正则表达式立即显示匹配结果
- **语法高亮** - 匹配结果高亮显示，便于查看
- **分组捕获** - 显示正则表达式的分组匹配结果
- **常用示例** - 内置邮箱、手机号、URL等常用正则表达式
- **标志支持** - 可视化选择g、i、m、s、u、y等正则标志

└── main.ts          # 入口文件

## 🛠 开发环境设置

### 环境要求

- Node.js >= 20.19.0 或 >= 22.12.0
- npm 或 yarn

### 快速开始

1. **克隆项目**

```bash
git clone <repository-url>
cd program-tool
```

2. **安装依赖**

```bash
npm install
```

3. **启动开发服务器**

```bash
npm run dev
```

4. **构建生产版本**

```bash
npm run build
```

5. **预览构建结果**

```bash
npm run preview
```

### 开发命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run preview` - 预览构建结果
- `npm run type-check` - 类型检查
